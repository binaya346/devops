# Prerequisites
Make sure you have installed Node and npm

## verify 
`node -v`
`npm -v`

# Installation

Go to the directory where you want to add react project. Run the following command:
`npm create vite@latest my-react-app`

=> Choose `React`
=> Choose `Javascript + React`
=> Choose `yes` for `npm`
=> Put `No` to experimental

# Running
Go inside the react project which will be named `my-react-app` if you have followed above command

`cd my-react-app`
Run: `npm run dev` This will run the react application at port `5173`

You can access react app using:
`http://localhost:5173`

# Building React
`npm run build`

# Running the build Application
`npm run preview`

Access the application at port `4173`
